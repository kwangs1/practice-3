# 개발환경
● Server OS : Windows10

● Language : JAVA 1.8

● WEB Server : Apache 

● WAS Server : Tomcat 9.0.11

● build tool : maven 2.5.1

● DB: Oralce XE 11g-winx64

● FrameWork : Spring 5.0.7.RELEASE

● ORM : mybatis 3.4.6

# 개발과정
● 회원가입

⤷ 이메일 인증 구현

⤷ (정규 표현식)유효성 검사

⤷ 회원 탈퇴 및 회원 정보 수정

● 다중 게시판 기능

● Ajax를 이용한 댓글 및 답글 CRUD

● Ajax를 이용한 게시글 및 댓글 좋아요, 싫어요
